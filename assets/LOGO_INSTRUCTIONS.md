# Asset Instructions

## Required Assets

This directory should contain the following image files:

### 1. logo.png
- **Size:** 1024x1024 pixels
- **Format:** PNG with transparency
- **Design:** Water droplet or wave icon with "AquaIntel" branding
- **Colors:** Use Indian flag colors (Saffron #FF9933, Blue #000080)
- **Purpose:** App icon and splash screen

### 2. ministry-logo.png
- **Size:** 512x512 pixels
- **Format:** PNG with transparency
- **Design:** Ministry of Jal Shakti official logo
- **Purpose:** Branding and credibility
- **Source:** https://jalshakti-ddws.gov.in/ (official logo)

## Temporary Placeholders

Until you add actual images, you can use:
- Online logo generators (e.g., Canva, LogoMaker)
- Icon libraries (e.g., Flaticon, IconFinder)
- Generate using AI tools (e.g., DALL-E, Midjourney)

## How to Add Images

1. Create or download the logo files
2. Rename them to `logo.png` and `ministry-logo.png`
3. Place them in this `/assets` directory
4. The app will automatically use them

## Quick Start Without Images

The app will work without images, but you'll see warnings in the console. For demo purposes, you can:
1. Use a solid color placeholder
2. Download free water-themed icons from https://www.flaticon.com/
3. Create simple designs using free tools like Canva

## Recommended Logo Design

**For logo.png:**
- Central element: Water droplet or wave
- Background: Gradient from Saffron to Blue
- Text: "AquaIntel" in modern sans-serif font
- Style: Flat design, modern, professional

**Color Scheme:**
- Primary: #FF9933 (Saffron)
- Secondary: #000080 (Navy Blue)
- Accent: #138808 (Green)
- Background: #FFFFFF (White)
